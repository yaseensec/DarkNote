# DarkNote

DarkNote - Python based simple text editor made using tkinter Module

Description
============

DarkNote is a simple text editor that emphasizes on simplicity.Only the essential features are implemented in the text editor.DarkNote defaults to Dark Theme and is simple to use

<img src="images/screenshot.png">

Features
=========

*3 Themes are available : Dark(default),light and monokoi<br />
*Find and Replace Functionality<br />
*Unlimited Undo/Redo Functionality<br />
*Status Bar which shows number of Characters and Words in the file<br />
*Font chooser,Font resize,Font color chooser Functionality<br />

Usage
======

Install Python3 and install python tkinter module<br />
Download icons folder and darknote.py file<br />
Execute darknote.py in Terminal using python-------$python darknote.py<br />

Keybindings
===========

Ctrl+n       : New<br />
Ctrl+o       : Open<br />
Ctrl+s       : Save<br />
Ctrl+Alt+s   : Save As<br />
Ctrl+q       : Quit<br />
Ctrl+z       : Undo<br />
Ctrl+Shift+z : Redo<br />
Ctrl+c       : Copy<br />
Ctrl+v       : Paste<br />
Ctrl+x       : Cut<br />
Ctrl+Alt+x   : Clear All<br />
Ctrl+a       : Select All<br />
Ctrl+f       : Find/Replace<br />
